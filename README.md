 🌤 SkyCast - Weather App

SkyCast is a modern and responsive weather application that provides real-time weather updates along with a 5-day forecast. It features dynamic UI themes based on weather conditions and allows users to switch between Celsius and Fahrenheit.

## 🚀 Features
- 🌍 Search weather by city
- 🌡 Real-time temperature, humidity, wind, and visibility
- 🔁 Toggle between °C and °F
- 🌤 Dynamic background based on weather conditions
- 📅 5-day weather forecast
- ⏳ Loading indicator and error handling

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## 📦 How to Run
1. Clone the repository
2. Open `index.html` in your browser

## ⚠️ Note
API key is required from OpenWeatherMap.
